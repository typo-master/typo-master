# Typo Master



# 一、目标

## 第一阶段目标  

用于从GitHub上寻找有拼写错误的并且有一些Star的项目来用于蹭PR。

## 第二阶段目标

用于从GitHub上寻找有待解决的Bug或者Feature的项目。 

## 第三阶段目标

对GitHub上的项目自动审计、Review，检出Bug或者漏洞进行修复。



此项目只负责完成第一阶段的目标，如果能成功完成，会有后续项目继续接力完成后续目标。







# 第一阶段的实现思路

## 项目的获取 

- 按照人的粉丝之类的去搜索寻找
- 按照标签搜索寻找

## 项目的检查

- 中文词典

- 英文词典
